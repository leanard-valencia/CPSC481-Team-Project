Team's Mission

Our team aims to create a project that intends to solve a real life problem and will be useful to many people. Futhermore, we would like to use this as an opportunity to improve and learn new skills for UI Design and to provide a project that enhances the user experience. Finally, our team intends to design and deliver tasks within the deadline and get a good grade for this project.
